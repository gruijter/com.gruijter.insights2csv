Exportieren Sie alle Ihre Insights auf Ihr NAS, einen FTP-Server oder einen WebDAV-kompatiblen Speicher. Die Insights werden gezippt und im .csv-Format gespeichert, sodass Sie sie in Excel öffnen und alle Arten von Statistiken und Diagrammen erstellen können. Der Export enthält auch eine JSON-Version der Daten.

Exporte können einfach mithilfe der Trigger-Karte in einem Flow geplant werden. Sie können einen vollständigen Export oder einen Export pro App planen.

Geben Sie zur Einrichtung das lokale Laufwerk (NAS) und/oder das FTP- und/oder das WebDAV-Informationen ein. Es ist möglich, eine lokale Freigabe, FTP und einen WebDAV-Speicher gleichzeitig zu verwenden. Wählen Sie 'Separaten Ordner für jeden Export', wenn Sie für jeden Export einen neuen Ordner erstellen möchten.

Hinweis: Das Ausführen eines Exports kann lange dauern (mehr als 10 Minuten), abhängig davon, wie viele Insights Sie haben. Es ist nicht möglich, die Daten in Homey wiederherzustellen.
