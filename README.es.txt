Exporte todos sus Insights a su NAS, a un servidor FTP o a un almacenamiento compatible con WebDAV. Los Insights se comprimen y almacenan en formato .csv para que pueda abrirlos en Excel y crear todo tipo de estadísticas y gráficos. La exportación también incluye una versión JSON de los datos.

Las exportaciones se pueden programar fácilmente mediante la tarjeta de activación en un flow. Puede programar una exportación completa o una exportación por aplicación.

Para configurar, introduzca la información del recurso compartido local (NAS), y/o FTP, y/o WebDAV. Es posible utilizar un recurso compartido local, FTP y almacenamiento WebDAV simultáneamente. Seleccione 'Carpeta separada para cada exportación' si desea crear una nueva carpeta para cada exportación.

Nota: hacer una exportación puede llevar mucho tiempo (más de 10 minutos), dependiendo de cuántos Insights tenga. No es posible restaurar los datos en Homey.
