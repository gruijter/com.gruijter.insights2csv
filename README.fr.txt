Exportez tous vos Insights vers votre NAS, un serveur FTP ou un stockage compatible WebDAV. Les Insights sont compressés et stockés au format .csv pour que vous puissiez les ouvrir dans Excel et créer toutes sortes de statistiques et graphiques. L'exportation inclut également une version JSON des données.

Les exportations peuvent être facilement planifiées à l'aide de la carte déclencheur dans un flow. Vous pouvez planifier une exportation complète, ou une exportation par application.

Pour configurer, entrez le partage local (NAS), et/ou les informations FTP, et/ou WebDAV. Il est possible d'utiliser un partage local, FTP et un stockage WebDAV simultanément. Sélectionnez 'Dossier séparé pour chaque exportation' si vous souhaitez créer un nouveau dossier pour chaque exportation.

Remarque : l'exportation peut prendre beaucoup de temps (plus de 10 minutes), selon le nombre d'Insights que vous possédez. Il n'est pas possible de restaurer les données dans Homey.
